Instructions

This is a Preview version of the Wordnik WordKit SDK for iOS and Mac OSX platforms. It contains the following components:

  • A Mac OSX Framework which includes a Data SDK for accessing the Wordnik API
  • An iOS Framework which includes a Data SDK as well as a UI SDK.
  • HTML Documentation

    • Please visit http://developer.wordnik.com to get an API key, which is required for use of these SDKs. In addition you can find code recipies, sample applications, and an interactive API shell for prototyping calls to the API
